# SGDE - Sistema de Gestão Eleitoral

O **SGDE** é um sistema desenvolvido em C para gerenciar processos eleitorais de forma automatizada, incluindo cadastro de candidatos, eleitores, votação e apuração de resultados.  
O objetivo é fornecer uma ferramenta didática e funcional para aprendizado de **Algoritmos e Estruturas de Dados** e gestão de eleições simuladas.

## Funcionalidades

- Cadastro de **eleitores** e **candidatos**
- Registro de **votos** e **validação de eleitor**
- **Apuração de resultados** de forma automática
- Utiliza **estruturas de dados** como listas, pilhas, filas e árvores para gerenciamento de informações

---

## Tecnologias utilizadas

- Linguagem: **C**
- Estruturas de dados aplicadas: **listas ligadas, pilhas, filas, árvores binárias**
- Ferramentas: **Git & GitHub** para controle de versão
